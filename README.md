HeimHafen Web ist meine persönliche Website für Webdesign & Entwicklung mit Fokus auf Campingplätze, Destinationen und familienfreundliche Betriebe.  
Die Seite zeigt mein Leistungsangebot, meinen Arbeitsansatz und verlinkt auf konkrete Demo-Projekte.

- **Live:** https://heimhafen.github.io/heimhafen-web-/
- **Tech-Stack:** Reines HTML, CSS und etwas JavaScript – bewusst ohne Frameworks, leichtgewichtig und schnell.
- **Schwerpunkte:** Klarer Nutzen für Kund:innen, Mobile-First, saubere Struktur für spätere Erweiterungen.
- **Einsatz:** Portfolio-Seite für Bewerbungen und als Landingpage für potenzielle Auftraggeber:innen.

## Aufbau

Die Seite besteht aus statischem HTML mit einer einseitigen Struktur (Single Page):

- Hero-Bereich mit Nutzenversprechen
- Zielgruppen und Leistungen
- Projekte / Showcases (Campingplatz & Café)
- Über mich / Arbeitsweise
- Ablauf der Zusammenarbeit
- Kontaktformular (derzeit ohne Backend-Anbindung)
