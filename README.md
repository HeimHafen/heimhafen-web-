# HeimHafen Web

HeimHafen Web ist meine persönliche Website für Webdesign & Entwicklung mit Fokus auf Campingplätze, Destinationen und familienfreundliche Betriebe.  
Die Seite zeigt mein Leistungsangebot, meinen Arbeitsansatz und verlinkt auf konkrete Demo-Projekte.

- Live: https://heimhafen.github.io/heimhafen-web-/
- Tech-Stack: Reines HTML, CSS und etwas JavaScript – ohne Frameworks, bewusst leichtgewichtig.
- Fokus: Klarer Nutzen für Kund:innen, mobil optimiert, saubere Struktur für spätere Erweiterungen.
- Einsatz: Portfolio-Seite für Bewerbungen und als Landingpage für potenzielle Auftraggeber:innen.
